Project Portfolio

Python Project - D&D Visual Database

In this project, I developed a Graphical User Interface (GUI) to display and search for spells from the popular table-top roleplaying game Dungeons & Dragons (D&D). I developed buttons, text format, a search bar and even an animated cursor completely from scratch. I retrieved my data from the official D&D 5th Edition API, which contains all the information regarding the attributes and characteristics for any asset in the game, in the JSON format. For this project, I relied on my skills in Python, but also gained exposure to the JSON text format, and knowledge in working with the Pygame library. I also applied the concepts of Object-Oriented Programming in the project, with extensive use of Polymorphism and Inheritance.

HackED C++ Project - MipsClips

MipsClips is designed to read and interpret MIPS Assembly Code, and emulate its execution in a manner akin to real life. In addition to the implementation of the core MIPS instruction types, the program features the ability to modify and view its own "simulated memory" system. For this project, I collaborated on a team with three of my friends in a 24-hour coding and design challenge that was presented to a panel of judges. This project taught me about memory allocation and optimization, and avoiding memory hazards, something that is vital to both C++ and Assembly Languages. It also gave me a practical experience in C++, as before this project my C++ skills felt diminished, but since the completion of this project I feel much more confident in my abilities.
